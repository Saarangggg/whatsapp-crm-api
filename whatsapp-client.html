✉️ Send Text/Media Message

Send custom plain-text messages or media files (images, PDFs, documents) via HTTP POST.

POST /api/send-message

Request Headers

Content-Type: application/json
x-wa-secret: your-secret-key

Request Payload (JSON)

{ "phone": "919048186012", "message": "Hello from the API!", "countryCode": "91" }

cURL Example

curl -X POST -H "Content-Type: application/json" \ -H "x-wa-secret: your-secret-key" \ -d '{"phone": "919048186012", "message": "Hello from the API!"}' \ http://localhost:3001/api/send-message

Node.js Fetch Example

fetch('http://localhost:3001/api/send-message', { method: 'POST', headers: { 'Content-Type': 'application/json', 'x-wa-secret': 'your-secret-key' }, body: JSON.stringify({ phone: '919048186012', message: 'Hello from Node!' }) }) .then(res => res.json()) .then(data => console.log(data));
